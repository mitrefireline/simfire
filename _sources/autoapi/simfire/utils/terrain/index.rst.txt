.. py:function:: random_seed_list(length, seed = None)

   Create a tuple of tuples  of random integers (to be used as seeds) based on `length`
   and initial input `seed`

   :param length: Length of each of the nested tuples.
   :param seed: Initial random seed for generating seeds.

   :returns: A tuple of tuples containing random integers ranging from 0 to 99_999

.. py:function:: chaparral(seed = None)

   Create a chaparral fuel object using an optional input seed

   The seed only affects the input to `w_0`, the oven-dry fuel load value.

   :param seed: Initial seed.

   :returns: A fuel with randomized `w_0`, `delta == 6.0`, `M_x == 0.2`, and `sigma == 1739`.
