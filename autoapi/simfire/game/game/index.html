update(terrain, fire_sprites, fireline_sprites, agent_sprites, wind_magnitude_map, wind_direction_map)#

Update the game display using the provided terrain, sprites, and environment data. Most of the logic for the game is handled within each sprite/manager, so this is mostly just calls to update everything.

Parameters:

  • terrain (simfire.game.sprites.Terrain) – The Terrain class that comprises the burnable area.

  • fire_sprites (Sequence[simfire.game.sprites.Fire]) – A list of all Fire sprites that are actively burning.

  • fireline_sprites (Sequence[simfire.game.sprites.FireLine]) – A list of all FireLine sprites that are dug.

  • wind_magnitude_map (Union[Sequence[Sequence[float]], numpy.ndarray]) – The map/array containing wind magnitudes at each pixel location

  • wind_direction_map (Union[Sequence[Sequence[float]], numpy.ndarray]) – The map/array containing wind directions at each pixel location
