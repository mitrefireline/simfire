reset_terrain(topography_seed=None, topography_type=None, fuel_seed=None, fuel_type=None, location=None)#

Reset the terrain functional generation seeds if using functional data, or reset the terrain lat/long location if using operational data.

Parameters:

  • topography_seed (Optional[int]) – The seed used to randomize functional topography generation

  • fuel_seed (Optional[int]) – The seed used to randomize functional fuel generation

  • location (Optional[Tuple[float, float]]) – A new center-point for the operational topography and fuel data

  • topography_type (Optional[str]) –

  • fuel_type (Optional[str]) –

Return type:

None

reset_wind(speed_seed=None, direction_seed=None)#

Reset the wind speed and direction seeds.

Parameters:

  • speed_seed (Optional[int]) – The seed used to randomize wind speed generation

  • direction_seed (Optional[int]) – The seed used to randomize wind direction generation

Return type:

None

reset_fire(seed=None, pos=None)#

Reset the fire initial position seed. Note that both seed and pos cannot be specified together since seed is used for random/dynamic cases and pos is used for static cases.

Parameters:

  • seed (Optional[int]) – The seed used to randomize fire initial position generation.

  • pos (Optional[Tuple[int, int]]) – The static position to start the fire at

Return type:

None
