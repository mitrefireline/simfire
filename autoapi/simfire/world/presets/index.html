simfire.world.presets#

This module contains preset FuelArray tile values for various terrain/fuel. Four significant figures were used for rounding For fuel load (w_0), the 1-hour fuel load is used For surface-area-to-volume (SAV) ratio (sigma), the characteristic SAV is used For M-x, the Dead fuel moisture of extinction is used Fir delta, the Fuel bed depth is used

As specified in: https://www.fs.fed.us/rm/pubs_series/rmrs/gtr/rmrs_gtr371.pdf

Urban, Snow/Ice, Agricutlture, Water, Barren are Non-Burnable fuel types as described in: https://gacc.nifc.gov/oncc/docs/40-Standard%20Fire%20Behavior%20Fuel%20Models.pdf
