WindNoise

Class for controlling and fine tuning wind noise generation with the Simplex noise
class simfire.world.wind_mechanics.perlin_wind.WindNoise(seed=None, scale=100, octaves=2, persistence=0.5, lacunarity=1.0)#

Class for controlling and fine tuning wind noise generation with the Simplex noise algorithm

Class that handles and creates the wind layer which specifies the magnitude and direction of wind a a given location. Uses python noise library

Parameters:

  • seed (Optional[int]) – The value to seed the noise generator

  • scale (int) – The “altitude” from which to see the noise

  • octaves (int) – number of passes/layers of the algorithm. Each pass adds more detail

  • persistence (float) – How much each pass affects the overall shape High values means each pass is less important on shape. Lower values mean each pass has greater effect on shape. Best to keep between 0-1

  • lacunarity (float) – Controls increase in frequency of octaves per pass. Frequency = lacunarity & (pass number). Higher lacunarity, higher frequency per pass.

  • screen_size – Size of screen (both heigh and width) MUST BE SQUARE
